# Scripts

### How To Run: 

```bash
curl -fsSL https://raw.githubusercontent.com/johnnymayodev/scripts/refs/heads/main/<path>/<script>.sh | bash
```

### Examples:

Update APT Packages

```bash
curl -fsSL https://raw.githubusercontent.com/johnnymayodev/scripts/refs/heads/main/apt/update.sh | bash
```

Install Docker with Docker Compose

```bash
curl -fsSL https://raw.githubusercontent.com/johnnymayodev/scripts/refs/heads/main/tools/docker/install.sh | bash
```
